# 今後のタスク・マイルストーン

本ドキュメントは、「暇つぶしシューティングゲーム」の今後の開発計画、追加予定の機能、および目指すべき方向性をまとめたものです。

## 🎯 ビジョン
**「プレイ中の楽しさを追求する」**
単なる暇つぶしにとどまらず、操作の気持ちよさ（Game Feel）、敵を倒した際の爽快感、スコアを稼ぐことへのモチベーションを最大化する。

---

## 📅 マイルストーンとタスク一覧

### Phase 1: プレイフィールの向上（楽しさの追求）
ゲームの手触りや視覚的・聴覚的なフィードバックを強化し、「動かしているだけで楽しい」状態を目指す。

- [ ] **エフェクトの追加**
  - 敵撃破時のパーティクル（破片）エフェクトの実装
  - 被弾時やボス撃破時の軽い画面揺れ（カメラシェイク）
- [ ] **コンボ（チェイン）システムの実装**
  - 敵を連続で倒すことでスコア倍率が上がるシステムの追加
  - ハイリスク・ハイリターンのアグレッシブなプレイスタイルを推奨する設計
- [ ] **敵のバリエーション強化**
  - 耐久力のあるシールド持ち敵や、分裂する敵などの新規ギミック追加

### Phase 2: 競技性とリプレイ性の強化
プレイヤーが繰り返し遊びたくなるような目標（ゴール）を提供する。

- [ ] **オンラインランキング機能の実装**
  - 累計スコアやクリアタイムを競うグローバルリーダーボード
  - データベース（Vercel KV, Supabase, Firebase等）の選定と連携
  - プレイヤー名の入力と表示機能
- [ ] **プレイ履歴と統計データ**
  - 過去のハイスコア推移、最大到達ステージ、倒した敵の総数などのローカル記録

### Phase 3: エンドコンテンツの追加
ゲームを一度クリアしたプレイヤー向けの、さらに歯応えのあるコンテンツを提供する。

- [ ] **エクストラステージ（裏面）の実装**
  - 全8ステージをクリアしたプレイヤー（`GAME_CLEAR` 達成者）のみが挑戦できる特殊モード
  - 初期難易度が大幅に上がり、敵の弾幕パターンが凶悪化する
  - エクストラ専用のボスや、さらに上位の武器強化（例: 5-Way、貫通レーザー）の解禁

### Phase 4: オンライン化（マルチプレイ・ソーシャル機能）
他プレイヤーとの繋がりを持たせ、ゲームの寿命をさらに延ばす。

- [ ] **リアルタイム協力プレイ（Co-op）の実装**
  - WebRTCやWebSocket（Socket.io等）を用いた2人協力プレイモード
  - 味方の回復や、協力して倒す巨大ボスの追加
- [ ] **オンライン対戦（スコアアタック）モード**
  - 同じステージ構成・同じ乱数シードで同時にスタートし、スコアを競うモード
- [ ] **ゴーストデータの共有**
  - ランキング上位者のリプレイ（ゴースト）をダウンロードし、一緒の画面でプレイする機能

---

## 📝 備考・アイデアメモ
- 現在はオフライン対応（PWA）がメインだが、ランキング機能を実装する場合はオンライン状態のチェックとキャッシュ戦略を検討する必要がある。
- スキル（必殺技）システムの導入も、プレイの楽しさ向上に寄与する可能性があるため要検討。
